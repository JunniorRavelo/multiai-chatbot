Elige tu proveedor de IA

Configura un proveedor en MultiAI ChatBot → AI Model. Los datos del visitante solo se envían cuando alguien usa el chat.

Recomendado

WordPress AI

wordpress_ai

  • Usa el AI Client de WordPress 7.0+
  • OpenAI, Google Gemini o Anthropic vía Connectors
  • Claves en Ajustes → Connectors
  • Modelo principal y fallback configurables
  • Fallback automático a Google si falla

Google IA

google_ia

  • Tu propia clave de Google Gemini
  • Llamadas directas a la API de Google
  • Modelos del catálogo Connectors
  • Clave en admin o wp-config.php
  • Modelo principal y fallback

Ollama

ollama

  • Modelos locales en tu infraestructura
  • Sin claves API en el plugin
  • URL por defecto: 127.0.0.1:11434
  • Ideal para privacidad y desarrollo
  • Ej.: llama3, mistral, gemma…

Panel de administración completo

Gestiona todo desde el menú MultiAI ChatBot en el escritorio de WordPress.

General AI Model Security Chat Style Statistics History
General Widget global, mensaje de bienvenida, system prompt, streaming e historial/estadísticas (desactivado por defecto).
AI Model Proveedor, modelos principal/fallback, clave Gemini (Google IA) y URL de Ollama.
Security Orígenes permitidos, caché, telemetría, límites por IP y suspensión por abuso.
Chat Style Presets visuales, colores personalizados, posición del widget y vista previa interactiva.
Statistics Totales, desglose por proveedor, latencia y exportación CSV.
History Conversaciones con ID CB-YYYY-MM-DD-HH-MM-SS, filtros y detalle de mensajes.

Instalación rápida

Genera el ZIP de producción sin archivos de desarrollo y súbelo a WordPress.

  1. Genera el paquete: ejecuta ./scripts/package-plugin en el repositorio.
  2. Sube el ZIP en Plugins → Añadir nuevo → Subir plugin (multiai-chatbot.zip).
  3. Activa el plugin desde la lista de plugins.
  4. Ve a MultiAI ChatBot y elige tu proveedor en la pestaña AI Model.
  5. Para WordPress AI, conecta proveedores en Ajustes → Connectors primero.
  6. Activa el widget en General y guarda. Si el streaming no responde, visita Ajustes → Enlaces permanentes y guarda de nuevo.
# Generar ZIP listo para WordPress
./scripts/package-plugin

# Verificar antes de publicar en WordPress.org
./scripts/verify-plugin-package

# Configuración opcional en wp-config.php
define( 'MULTCH_PROVIDER', 'google_ia' );
define( 'MULTCH_GEMINI_API_KEY', 'tu-clave' );
define( 'MULTCH_GEMINI_MODEL', 'gemini-2.5-flash' );
